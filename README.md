# EnGine_v4
=====================================================
AUTHOR:  Eran Negrin (eranegrin@gmail.com)
=====================================================

A game engine, written in C++ and OpenGL.
* Rigid body simulation using PhysX 3.1 sdk 
* Models and animation loading using FBX 2016.1.2 sdk.
* Stereo rendering.
* UI manager
* OpenGL 4 and above
